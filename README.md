## Features
* View your and others profiles
* Follow your class schedule
* View info about projects
* Look for vacancies in Project Tinder
* Apply for vacancies

## Stack
* Kotlin
* Coroutines
* Clean Architecture
* MVP
* Multimodule Architecture
* Dynamic Features
* Moxy
* Jetpack Navigation
* Glide
* Dagger
* Retrofit
* Timber
